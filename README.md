## Installation

- Copy default_config.json to config.json and set all required parameters
- Install Node (v8.11.x)
- Install all packages via "npm install"

## Usage

- Copy default_upload.csv to upload.csv
- Setup upload.csv
- Close upload.csv !Important
- Run 'node main.js'
